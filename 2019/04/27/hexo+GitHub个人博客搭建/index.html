流程

  • 创建一个名为 “用户名.github.io”的仓库

    • 如：changyeyi.github.io

    • 点击setting，往下滑，可以看到Github Pages的介绍说明

      • you site is published at https://changyeyi.github.io/

  • 添加SSH keys 到github

    • 生成本机电脑SSH keys

      • 配置配置全局name和email

      • 生成ssh keys

        • ssh-keygen -t rsa -C “changyeyi@outlook.com“，然后全部回车即可

        • 生成的ssh key会在当前用户的.ssh中的id_rsa.pub文件中

    • 添加key到github

博客部署

  • 安装用于部署到github上的插件，注意要将路径切换到前面初始化的文件夹(hexo)后执行以下命令

    • npm install hexo-deployer-git –save

  • 修改仓库地址

    • 初始化的文件夹中，有一个_config.yml的文件，修改其中内容

  • hexo deploy发布

  • 访问https://changyeyi.github.io 即可看到hexo提供的一个hello world实例页面
